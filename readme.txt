This program was used to chrome some items a bunch of times

usage:

chrome an item
node chrome.js <type> <times> <file>
  type can be
    1: in a currency tab, with the item in the center slot
    2: in a crafting bench, with the topmost option being the one it will use
    3: in a crafting bench, with the cursor hovered over the option
  times is the number of times to repeat it
  file is the name of the filename to write to. If it already exists it will be appended to

print current mouse coords
node mouse

analyze a data file
node analyze <file>
